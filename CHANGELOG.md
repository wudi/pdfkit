# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2025-11-22

> **Note:** This project is currently in an experimental phase.

### Added
- Initial release of PDFKit.
- Full support for PDF 1.0 - 2.0 specifications.
- Implementation of PDF/A, PDF/X, PDF/UA, PDF/VT compliance standards.
- Comprehensive support for color spaces, patterns, shading, and transparency.
- Font support for TrueType, Type 1, Type 3, OpenType, and CID-Keyed fonts.
- Interactive features including Annotations, Actions, and Forms (AcroForms, XFA).
- Security features: Encryption (RC4, AES) and Digital Signatures (PAdES, LTV).
- Structure and Metadata support: Tagged PDF, XMP, GeoSpatial.
