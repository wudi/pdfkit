# Examples

The `examples/` directory in the root of the repository contains complete working examples.

## Available Examples

- **`examples/merge/`**: Combine multiple PDF files into a single document.
- **`examples/split/`**: Split a PDF into individual pages or ranges.
- **`examples/extract_text/`**: Extract text content from a PDF.
- **`examples/watermark/`**: Add a watermark to existing PDF pages.
- **`examples/invoice/`**: Generate a complex invoice document from scratch.
- **`examples/cjk/`**: Handle Chinese, Japanese, and Korean fonts.
- **`examples/fonts/`**: Font embedding and subsetting examples.

## Running Examples

You can run any example using `go run`. For instance:

```bash
cd examples/invoice
go run main.go
```

Check the source code in each example folder for detailed implementation patterns.
