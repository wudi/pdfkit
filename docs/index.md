# PDFKit Documentation

Welcome to the documentation for **PDFKit**, a comprehensive, pure Go library for PDF manipulation.

## Overview

PDFKit is designed to support the full spectrum of PDF standards from PDF 1.0 to PDF 2.0, including PDF/A, PDF/X, PDF/UA, and PDF/VT.

## Documentation Sections

- [Getting Started](getting_started.md): Installation and basic usage.
- [Architecture](architecture.md): Detailed design and architecture of the library.
- [API Reference](api_reference.md): Overview of the packages and their responsibilities.
- [Examples](examples.md): Common usage patterns and examples.

## Features

- **Core Syntax**: Full read/write support for PDF 1.0 - 2.0.
- **Graphics**: Advanced color spaces, patterns, shading, and transparency.
- **Fonts**: TrueType, Type 1, OpenType, CID-Keyed fonts, and subsetting.
- **Forms**: AcroForms, XFA, and HTML Forms.
- **Compliance**: PDF/A, PDF/X, PDF/UA validation and enforcement.
- **Security**: Encryption (RC4, AES) and Digital Signatures (PAdES, LTV).
